#                 湖南农业大学本科毕业论文 LaTeX 模板

这个目录包含一个按附件 Word 说明模板整理的 LaTeX 模板工程，重点复刻了以下版式要求：

- 封面：上、下 2 cm，左、右 2.6 cm；校名使用一号行楷，论文类型使用一号黑体，题目使用小二黑体。
- 目录：1.5 倍行距；上、下 2.54 cm，左、右 2.6 cm；目录标题为二号黑体。
- 正文：上、下 2.54 cm，左、右 2.6 cm；中文小四号宋体，英文 Times New Roman；固定值约 22 磅行距。

## 文件说明

- `hnau-thesis.cls` ：模板核心类文件。
- `hnau-latex` ：论文 LaTeX 模板

## 说明

- 当前模板依据湖南农业大学毕业论文 Word 模板制作，主要是将学校规定的格式转化成latex模板格式。
- 如果学院还有更细的格式补充，可以继续在 `hnau-thesis.cls` 中调整。 
- 快毕业了不想给论文排版，纯分享项目，方便本校同学的论文撰写。
